# Отчёты и дашборд

## Разделы дашборда

Дашборд доступен по адресу `/dashboard` для пользователей с ролями `director` и `admin`.

Состоит из двух вкладок:
1. **Детальный отчёт** — таблица всех работ
2. **Сводный отчёт** — агрегированные данные

---

## 1. Детальный отчёт (`/dashboard`)

### Описание

Полная таблица со всеми записями о работах. Аналог "реестра документов".

### Фильтры

| Фильтр | Тип | Описание |
|--------|-----|----------|
| Период с / по | Date picker | Диапазон дат (`work_date`) |
| Корпус | Multi-select | Один или несколько корпусов |
| Вид работы | Multi-select | Один или несколько видов работ |
| Подрядчик | Multi-select | Один или несколько подрядчиков |
| Поиск | Text input | Поиск по описанию работы |

### Колонки таблицы

| Колонка | Описание | Редактируемо |
|---------|----------|--------------|
| № | Порядковый номер | ❌ |
| Дата | Дата выполнения работы | ❌ |
| Корпус | Номер и название корпуса | ❌ |
| Подрядчик | ФИО подрядчика | ❌ |
| Вид работы | Наименование вида работы | ❌ |
| Наименование работы | Описание | ❌ |
| Ед. изм. | Единица измерения работы | ❌ |
| Кол-во | Количество по работе | ❌ |
| Цена работы | Цена за единицу работы | ✅ Только director |
| Сумма работы | Итог по работам | ❌ (авто) |
| Материалы | Список использованных материалов | ❌ |
| Сумма материалов | Итог по материалам | ❌ (авто) |
| **ИТОГО** | **Общая сумма** | ❌ (авто) |
| Фото | Количество фото / просмотр | ❌ |
| Файлы | Количество файлов / скачать | ❌ |
| Дата создания | Когда внесена запись | ❌ |

### Пример строки

| Дата | Корпус | Подрядчик | Вид работы | Наименование | Ед.изм. | Кол-во | Цена | Сумма работы | Материалы | Сумма мат. | ИТОГО |
|------|--------|-----------|------------|--------------|---------|--------|------|--------------|-----------|------------|-------|
| 27.05.2025 | 12 | Иванов И.И. | Монтаж трубы | Замена трубопровода | м | 15.5 | 350.00 | 5 425.00 | Труба ПП 20мм (15.5м), Фитинг (3шт) | 3 632.93 | 9 057.93 |

### Пагинация

- По умолчанию: 50 строк на страницу
- Варианты: 25, 50, 100, 200
- Всего записей отображается над таблицей

### Итоги по таблице (подвал)

```
Всего записей: 250
Сумма по работам: 125 000.00 ₽
Сумма по материалам: 89 000.00 ₽
ИТОГО: 214 000.00 ₽
```

### Выгрузка в Excel

**Кнопка:** "📥 Выгрузить в Excel"

**Формат файла:**
- Имя: `works_detailed_2025-05-01_2025-05-27.xlsx`
- Лист: "Детальный отчёт"
- Шапка закреплена
- Автоширина колонок
- Формат чисел: число с 2 знаками после запятой
- Итоги внизу таблицы (жирным)

## 2. Сводный отчёт (`/dashboard/summary`)

### Описание

Агрегированные данные с группировкой. Позволяет быстро оценить объёмы работ и затраты.

### Варианты группировки

| Группировка | Описание |
|-------------|----------|
| **По корпусам** | Сколько работ и на какую сумму по каждому корпусу |
| **По видам работ** | Сводка по каждому виду работы |
| **По датам** | Группировка по дням (или месяцам, если период большой) |
| **По подрядчикам** | Сводка по каждому подрядчику |

### Фильтры

| Фильтр | Тип | Описание |
|--------|-----|----------|
| Период с / по | Date picker | Обязательный фильтр |
| Корпус | Multi-select | Опционально |
| Группировка | Radio buttons | Выбор одного из вариантов |

### Колонки таблицы (при группировке "По корпусам")

| Колонка | Описание |
|---------|----------|
| Корпус | Номер корпуса |
| Кол-во работ | Сколько записей о работах |
| Кол-во по работам | Суммарное количество (м, шт и т.д.) |
| Сумма по работам | Σ (цена работ × количество) |
| Сумма по материалам | Σ (цена материалов × количество) |
| **ИТОГО** | **Общая сумма** |

### Пример (группировка по корпусам)

| Корпус | Кол-во работ | Сумма работ | Сумма материалов | ИТОГО |
|--------|--------------|-------------|------------------|-------|
| 12 | 45 | 75 000.00 | 52 000.00 | 127 000.00 |
| 15А | 32 | 50 000.00 | 37 000.00 | 87 000.00 |
| **ИТОГО** | **77** | **125 000.00** | **89 000.00** | **214 000.00** |

### Пример (группировка по видам работ)

| Вид работы | Ед.изм. | Кол-во работ | Общее кол-во | Сумма | Доля |
|------------|---------|--------------|--------------|-------|------|
| Монтаж трубы | м | 25 | 387.5 | 135 625.00 | 63% |
| Установка счётчика | шт | 8 | 8 | 9 600.00 | 4% |

### Пример (группировка по датам)

| Дата | Кол-во работ | Сумма работ | Сумма материалов | ИТОГО |
|------|--------------|-------------|------------------|-------|
| 27.05.2025 | 5 | 8 500.00 | 6 200.00 | 14 700.00 |
| 26.05.2025 | 3 | 5 000.00 | 3 800.00 | 8 800.00 |

**Автоматический выбор периода группировки:**
- Диапазон ≤ 31 день — группировка по дням
- Диапазон 32-365 дней — группировка по месяцам
- Диапазон > 365 дней — группировка по годам

### Пример (группировка по подрядчикам)

| Подрядчик | Кол-во работ | Сумма работ | Сумма материалов | ИТОГО |
|-----------|--------------|-------------|------------------|-------|
| Иванов И.И. | 30 | 45 000.00 | 32 000.00 | 77 000.00 |
| Петров П.П. | 25 | 38 000.00 | 28 000.00 | 66 000.00 |

### Выгрузка в Excel

**Кнопка:** "📥 Выгрузить сводный отчёт"

**Формат файла:**
- Имя: `works_summary_by_building_2025-05-01_2025-05-27.xlsx`
- Лист: "Сводный отчёт"
- Шапка закреплена
- Итоги по колонкам
- Дополнительный лист с графиком (если группировка по датам/корпусам)

## 3. Редактирование цен (только director)

### Описание

Заместитель директора может корректировать цены непосредственно в таблице детального отчёта.

### Механизм

1. Director открывает детальный отчёт
2. Находит нужную строку
3. Кликает по ячейке "Цена работы" или "Цена материала"
4. Ячейка становится редактируемой (inline editing)
5. Вводит новую цену
6. Нажимает Enter или кликает вне ячейки
7. Система отправляет запрос на сервер
8. Суммы пересчитываются автоматически

### Пример inline editing

```
До редактирования:
┌──────────┐
│ 350.00   │ ← клик
└──────────┘

Редактирование:
┌──────────┐
│ 400.00 │  │ ← input с текущим значением
└──────────┘

После сохранения:
┌──────────┐
│ 400.00 ✓ │ ← обновлено, суммы пересчитаны
└──────────┘
```

### Ограничения

- Редактировать может только пользователь с ролью `director`
- Можно редактировать только цены (unit_price работы и материалов)
- Количество, дата, корпус и прочие поля **не редактируются**
- История изменений цен **не ведётся** (только updated_at в записи)
- При изменении цены автоматически пересчитываются все зависимые суммы

### API

```
PUT /api/works/{id}/prices

Request:
{
  "service_unit_price": 400.00,
  "materials": [
    {
      "material_id": 12,
      "unit_price": 250.00
    }
  ]
}

Response:
{
  "service_total_price": 6200.00,
  "materials_total_price": 4143.43,
  "total_price": 10343.43
}
```

## 4. Печатная форма

### Описание

Возможность сформировать печатную версию отчёта для подписи.

### Функционал

- Кнопка "🖨️ Печать" в детальном отчёте
- Открывается новая вкладка с версией для печати
- Убраны лишние элементы UI (кнопки, фильтры)
- Добавлены:
  - Логотип организации (placeholder)
  - Название отчёта
  - Период
  - Дата формирования
  - Подпись: "Зам. директора по АХЧ _______________"

## 5. Экспорт данных

### Поддерживаемые форматы

| Формат | Детальный отчёт | Сводный отчёт |
|--------|-----------------|---------------|
| Excel (.xlsx) | ✅ | ✅ |
| PDF | ❌ (v2) | ❌ (v2) |
| CSV | ❌ | ❌ |

### Настройки экспорта

При нажатии "Выгрузить в Excel":
1. Система применяет текущие фильтры
2. Экспортирует только отфильтрованные данные
3. Файл автоматически скачивается
4. Имя файла содержит даты периода и тип отчёта
