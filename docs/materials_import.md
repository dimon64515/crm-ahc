# Импорт справочников из Excel

## Общее описание

Администратор загружает Excel-файлы (.xlsx) для обновления справочников:
- **Материалы** — `materials`
- **Виды работ / услуги** — `services`

Импорт производится через раздел настроек (`/settings/materials` и `/settings/services`).

---

## Формат файла «Материалы»

### Структура

| Колонка A | Колонка B | Колонка C |
|-----------|-----------|-----------|
| **Наименование материала** | **Единица измерения** | **Цена за единицу** |

### Пример файла

```
Наименование материалов применяемых при проведении работ | Единица измерения | Цена за единицу
------------------------------------------------------------------------------------------------------------
Труба из полипропилена армированная стекловолокном 20 мм | Метр              | 217,06
Труба из полипропилена армированная стекловолокном 25 мм | Метр              | 285,50
Фитинг угловой 20 мм                                      | шт                | 89,50
Фитинг тройник 20 мм                                      | шт                | 120,00
Кран шаровой 20 мм                                        | шт                | 450,00
```

### Правила

- **Первая строка** — заголовки (игнорируется при импорте)
- **Наименование** — уникальное, обязательное поле (макс. 255 символов)
- **Единица измерения** — текст, опционально (макс. 50 символов)
- **Цена** — число, обязательное. Разделитель: запятая `,` или точка `.`
- **Пустые строки** — пропускаются
- **Дубликаты по названию** — обновляют существующую запись (цена и ед.изм.)

---

## Формат файла «Виды работ / Услуги»

### Структура

| Колонка A | Колонка B | Колонка C |
|-----------|-----------|-----------|
| **Наименование товара, услуги (работы)** | **Единица измерения** | **Цена за единицу** |

### Пример файла

```
Наименование товара, услуги (работы) | Единица измерения | Цена за единицу
-------------------------------------------------------------------------------------------------
Монтаж трубы полипропиленовой         | м                 | 350,00
Демонтаж старой трубы                 | м                 | 150,00
Установка счётчика воды               | шт                | 1200,00
Замена крана                          | шт                | 500,00
Прокладка электрокабеля               | м                 | 80,00
```

### Правила

- Идентичны правилам для материалов
- Отдельный справочник `services`

## Процесс импорта

### 1. Загрузка файла

```
[Админ] ──► [Выбирает файл .xlsx]
              │
              ▼
         [Валидация]
              ├── Формат: .xlsx
              ├── Размер: ≤ 5 МБ
              └── Структура: минимум 3 колонки
              │
              ▼
         [Сохранение]
              └── Временный файл в /uploads/temp/
```

### 2. Чтение и парсинг

```python
# Псевдокод процесса
import pandas as pd

df = pd.read_excel(file_path, header=0)
# Ожидаемые колонки: [0] - Наименование, [1] - Ед.изм., [2] - Цена

results = {
    'total_rows': len(df),
    'created': 0,
    'updated': 0,
    'errors': [],
    'skipped': 0
}

for index, row in df.iterrows():
    name = str(row[0]).strip()
    unit = str(row[1]).strip() if pd.notna(row[1]) else None
    price_str = str(row[2]).strip().replace(',', '.').replace(' ', '')
    
    # Валидация строки
    if not name:
        results['errors'].append(f"Строка {index + 2}: пустое наименование")
        continue
    
    try:
        price = float(price_str)
        if price < 0:
            raise ValueError()
    except:
        results['errors'].append(f"Строка {index + 2}: некорректная цена '{price_str}'")
        continue
    
    # Поиск существующей записи
    existing = db.query(Model).filter(Model.name == name).first()
    
    if existing:
        existing.unit = unit
        existing.price = price
        existing.is_active = True
        existing.updated_at = now()
        results['updated'] += 1
    else:
        db.add(Model(name=name, unit=unit, price=price))
        results['created'] += 1

db.commit()
```

### 3. Обработка дубликатов

| Сценарий | Действие |
|----------|----------|
| Наименование совпадает | Обновить цену и ед.изм., активировать |
| Наименование новое | Создать запись |
| Наименование пустое | Пропустить, зафиксировать ошибку |
| Цена некорректная | Пропустить строку, зафиксировать ошибку |

### 4. Результат импорта

**Успешный импорт:**
```json
{
  "success": true,
  "data": {
    "total_rows": 500,
    "created": 120,
    "updated": 380,
    "errors": 0,
    "message": "Импорт завершён успешно"
  }
}
```

**Импорт с ошибками:**
```json
{
  "success": true,
  "data": {
    "total_rows": 500,
    "created": 118,
    "updated": 379,
    "errors": 3,
    "error_details": [
      "Строка 45: пустое наименование",
      "Строка 128: некорректная цена 'договорная'",
      "Строка 256: отрицательная цена '-100'"
    ]
  }
}
```

## Ограничения

| Параметр | Значение |
|----------|----------|
| Максимальный размер файла | 5 МБ |
| Максимум строк | 10 000 |
| Формат | .xlsx (Excel 2007+) |
| Не поддерживается | .xls, .csv, .ods |
| Кодировка | UTF-8 (автоопределение) |

---

## Обработка ошибок

### Ошибки файла

| Ошибка | Решение |
|--------|---------|
| Файл слишком большой (> 5 МБ) | Разбить на несколько файлов |
| Неподдерживаемый формат | Сохранить как .xlsx |
| Отсутствуют колонки | Проверить структуру (нужно минимум 3 колонки) |

### Ошибки данных

| Ошибка | Решение |
|--------|---------|
| Пустое наименование | Заполнить ячейку |
| Некорректная цена | Использовать число (с точкой или запятой) |
| Отрицательная цена | Использовать положительное число |
| Дубликаты внутри файла | Оставить одну строку, удалить остальные |

## Логирование

Все операции импорта логируются:

```sql
CREATE TABLE import_logs (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    import_type VARCHAR(20),      -- 'materials' или 'services'
    filename VARCHAR(255),
    total_rows INTEGER,
    created_count INTEGER,
    updated_count INTEGER,
    error_count INTEGER,
    error_details JSONB,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
