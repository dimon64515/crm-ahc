# CRM для заместителя директора по АХЧ

Кастомная CRM-система для учёта работ подрядчиков на объектах (корпусах), управления справочниками материалов и услуг, формирования отчётности.

## Домен

`https://report.fanat-mv.ru`

## Стек технологий

- **Frontend:** React 18 + Vite + React Router
- **Backend:** Python 3.11 + FastAPI + SQLAlchemy
- **База данных:** PostgreSQL 15
- **Хранилище файлов:** Локальная файловая система (организованная по папкам корпусов)
- **Веб-сервер:** Nginx (reverse proxy)
- **Аутентификация:** JWT (JSON Web Tokens)

## Роли пользователей

| Роль | Описание |
|------|----------|
| **Подрядчик** | Заполняет форму выполненных работ, прикрепляет фото и файлы. Не может редактировать цены. |
| **Директор (зам. АХЧ)** | Просматривает дашборд, фильтрует данные, выгружает отчёты в Excel, редактирует цены в отчётах. |
| **Администратор** | Управляет пользователями, загружает/обновляет справочники материалов и услуг через Excel. |

## Основные модули

1. **Авторизация и управление пользователями**
2. **Форма подрядчика** — внесение данных о выполненных работах (фото сжимаются до 1 МБ)
3. **Справочники** — импорт материалов и видов работ из Excel
4. **Дашборд директора** — таблица всех работ с фильтрами и выгрузкой
5. **Сводный отчёт** — агрегированные данные по корпусам, видам работ, суммам
6. **Бэкапы** — архивация фото и полных данных системы с возможностью восстановления

## Быстрый старт (будет дополнено)

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8090

# Frontend
cd frontend
npm install
npm run dev
```
